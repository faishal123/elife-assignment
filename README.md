
## Run Locally

Clone the project

```bash
  git clone https://github.com/faishal123/elife-assignment.git
```

Go to the project directory

```bash
  cd elife-assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
